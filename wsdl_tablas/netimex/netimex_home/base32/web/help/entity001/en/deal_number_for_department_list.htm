Deal Number for Department 

Overview

NetIMEX allows for the automatic generation of the host deal number at the time of transaction initiation.  The deal number is generated prior to sending it to the host and receiving a return acknowledgement.

The deal number generation is based on the following logic:

  • Each customer is assigned a department ID, per deal type as part of the customer's details, which are downloaded, from the host.
  • For each department NetIMEX maintains a per deal type numerator.
  • At transaction initiation the relevant department per deal type numerator is incremented accordingly.

This function is used to setup the initial value of the numerator. Click on the Deal Number for Department link.  The list of departments is displayed. The list includes the department numbers and names.
