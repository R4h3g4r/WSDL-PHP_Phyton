Preferences

NetIMEX allows the user to customize the user’s general details and configure the appearance of the Inbox. The Inbox Settings gives the user control over opening and closing the four Inbox sections by default.

By clicking on the Customize Preferences option on the Customize List, the Customize Users Form is displayed.

User Details:

User ID - A unique description that is assigned to each user.

User Name - Full name of the User.

Default Customer - This is the customer on which the user will work on his behalf when entering the system. In order to enter any transaction on behalf of other customers, the user must return to the Choose Customer field by clicking on the Customer link on the top of each page. The Choose Customer field allows the user to switch to another customer’s transactions.

Default Language - After designating a default Language in the Languages Page of the Customize section, that language appears whenever the system is opened. If the user chooses another language as the default, the system will then appear in that language.

Super User - By checking this box the user may release any transaction for transmission by signing it. The user has the authorization to make the final signature on any transaction, regardless of pending signatures usually required for the specific transaction.

Supervisor - When this box is checked it means that the user has the authorization of Supervisor. A supervisor can assign authorizations to other users.

Group Inquiries - Checking this checkbox will allow the user to run an inquiry upon all the customers that have been assigned to him. This is relevant for the following:

  • Deal Select
  • Transaction Status
  • Open Deals
  • Deal Report

Inbox Settings:

View Items - Click here to view the Inbox as top down format, item by item.

  • View Tables - Allows you to view the Inbox in a table.  Each Inbox item in its respective section is displayed in one line, increasing the number of items visible in the screen.
  • Open Incoming Correspondence – When checked this Inbox section will be open unless the Display Closed Inbox After Transaction checkbox is checked.
  • Open Payment/Acceptance – When checked this Inbox section will be open unless the Display Closed Inbox After Transaction checkbox is checked.
  • Open Signature Required – When checked this Inbox section will be open unless the Display Closed Inbox After Transaction checkbox is checked.
  • Open Transactions in Process – When checked this Inbox section will be open unless the Display Closed Inbox After Transaction checkbox is checked.
  • Display Closed Inbox After Transaction – Indicates the Inbox appearance after handling a transaction. When unchecked (default) it behaves in the original functionality and leaves the Inbox open, using the setting as defined by the other indicators. When checked, all the Inbox sections are displayed closed.
