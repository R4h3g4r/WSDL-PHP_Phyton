PO Integration

The integration of NetIMEX with PO systems is specifically tailored for each Bank.

One of the integration functions is the ability to send details of NetIMEX templates to the PO system so it can generate L/C applications based on a specific template. The forwarding of the template details to the PO system is a background ongoing process.

This function allows the retransmission of the template details to the PO system.

In order to send NetIMEX templates to the PO system:

  • Click on the PO Integration link on the Bank Maintenance page to display the Bulk Templates Transfer Request Window.
  • Select a Customer or ‘All’ for the whole list of customers.
  • Click on the Send button.
