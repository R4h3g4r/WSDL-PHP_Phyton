AMEND TRANSFER LETTER OF CREDIT STEP (TAM)

Overview

The Transfer Amendment step (TAM) is used to make any amendments required to a Transfer Letter of Credit. 

Upon selection of the Amend Transfer L/C option on the Export Menu, the following Search Form is displayed:

The user can select a specific transaction by either:

  • Entering the Deal Number and/or Reference Number of the Transfer L/C that needs to be amended and clicking the Search button on the Search Form; or

  • By leaving the Deal Number and Reference Number fields empty and clicking the Search button on the Search form to display a list of relevant transactions. The user can then select a specific transaction from the list.

To view the deal, click on the deal title, which is blue and underlined. No changes can be made to the deal from the view mode.

To retrieve a Transfer L/C in order to amend it, click the Open icon next to the Deal you wish to amend. The selected deal is displayed its form completed.

Clicking on any heading in the navigation toolbar from anywhere in the application form will bring you to that section. There are three sections that must be completed in order to amend a TFR:

  • General Details

  • Settlement Data

  • Text
