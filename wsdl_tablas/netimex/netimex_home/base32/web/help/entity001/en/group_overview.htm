Group

A Group consists of a super organization that may contain several subdivisions that we define as Customers. Users belong to Groups and are authorized to enter transactions at the Bank on behalf of specific Customers.

Click on the Group link on the Bank Maintenance Option Page to display the list of Groups and their customers. In this section the user can define and edit the groups, and their users.

Group ID – This is the Group ID code by which the group is identified within the system.

Group Name – This is the full Group name.

Customer Name – This link allows the user to view the customers within the group.

Delete – Clicking on this icon erases the group from the system.

Edit – This function allows the user to make changes to the customer, users or group details.
