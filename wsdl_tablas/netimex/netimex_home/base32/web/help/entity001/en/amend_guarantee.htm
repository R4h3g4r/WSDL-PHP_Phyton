AMEND GUARANTEE

Amending a Guarantee Application

When viewing or editing a saved transaction, after the first save the amended fields (in relation to the previous version of the deal) are highlighted.

Navigation Tab

Clicking on any heading in the Navigation Toolbar from anywhere in the application form will bring you to that section.

The Direction Arrow or Page Numbers allow you to move through the Amend deals.

There are two sections of the Guarantee that may be amended.

  • General Details

  • Text

General Details

Clicking on the General Details heading in the Navigation Toolbar from anywhere in the application form will bring you to this section. 

To complete the General Details section, enter the relevant fields. Some of these fields are mandatory. They are indicated in Bold on the screen and by this icon  in this User Guide.

The General Details section requests the following information:

Reference Number - A unique deal identification number, decided by the NetIMEX user. For example, a contract number, a delivery order, etc.

    Amount - The amount of the guarantee Application.

    Currency - The currency in which the Application is expressed.

Deal Number - The bank specifies the deal number once the guarantee Application has been received. When the application is returned, the deal number appears.  This field is a read only field.

Guarantee Type - The type of guarantee.  Guarantee types are predefined by the bank. Examples include Performance, Bid, or Payment Guarantees.

Expiry Date - Specifies the date the guarantee expires. Once the date has passed, he guarantee is no longer valid.

Send Guarantee by - specifies which mode of transmission will be used to send the guarantee to the beneficiary or advising bank. Choices include by airmail, courier, mail, SWIFT, telex etc.

Text

The second part of the Guarantee form contains the Participants section. Scroll down from the General Details to access this section. This window enables you to enter additional information relevant to the Guarantee and enter account details.  Additional information can be entered for these categories and may include other custom‑defined categories.

To modify any additional deal instructions, select a category from the instruction list and modify the free format text as appropriate.

You may also access the Text section from anywhere in the application form. Clicking on the Text heading of the yellow Navigation Toolbar that precedes each section will bring you to this section.

This section enables you to enter additional information relevant to the Guarantee and enter account details.  Additional information can be entered for the following categories:

Customer Instructions - Used to enter additional customer instructions. To enter other deal information, click the appropriate option in the toolbar, or click the directional button to view a preceding window.

Guarantee Text - Used to enter additional stipulations or conditions to the Guarantee agreement. Enter additional instructions in the free format text window.

Zoom Text Icon  - Click on this icon to enlarge the text box area. You may edit or enter new data in the enlarged text box.

Open Clause Library Icon  - When clicking on this icon a list of defined clauses relevant for the window are displayed. After selecting the required clauses the contents of the clause is copied to the window.

Clause Library
In order to display the clauses for a specific text window click on the Open Clause Library Icon next to the relevant window.

On the Clause Library window select the clause/s that should be entered into the text by checking the relevant checkbox and clicking on the Select button. In order to include all the Clauses listed in the Clause Library check the checkbox at the top of the column and click on the Select button.

When returning to the Text Screen the user can view the detailed description of the Clause which will be included in the Text. In order to remove a clause from the Text the user must delete the text of the clause from the relevant window.

Defined Clause Descriptions
An additional option exists to display clauses as defined in the clauses form, which the user can select by selecting the desired checkboxes. This replaces the use of the Clause Library and free text windows.

There are two options for this function, which is established during bank setup:

  • The user can select more than one clause from one text window.

  • The user can select only one clause from one text window.

In a text window which is marked as mandatory, a validity check is performed to ensure that at least one clause is selected.

Account Details

The Customer Account Details Window enables you to instruct the bank how to make payment related to the Guarantee. 

Up to three Bank Charges can be defined. If data exists in the Payment Item field a corresponding Account number must be entered. If there is a blank Payment Item field, when relevant, any additional Payment Item and corresponding Account number may be selected.

To specify the customer’s account details, select the appropriate field on the Account Instructions Form. The Bank Charges describes the payment item to be paid, for example, Principal Amount, Charges. The Account Number selected from the drop-down list is from the Customer Profile. Use the Additional Instructions window to add additional payment instructions.

Additional Information

This section includes additional details of the transaction. If a field header is bold it indicates that the field is mandatory and the user must enter the relevant text in the corresponding data entry field. If the field header is not bold it is not mandatory and the user does not have to enter data.

Save and Authorize the Amended Guarantee

The following commands are located at the bottom of the Import Guarantee window.

Click here to save the data to the Amend Guarantee application.

When the application is completed, it must be signed. Click here to sign the application.

This button returns the user to the previous page without saving any action on the current page.

To enter other deal information, click the appropriate option in the toolbar, or click the directional button to view a preceding window.

When all the details have been entered into the application form, click on the Sign button. The system will run internal checks to make sure the information is entered correctly. If any errors exist, you will be notified about each error and given an opportunity to correct them.
