Customize Parties

Parties

The Parties Profile window lists the parties (deal participants) that have a dealing relationship with the customer and allow maintenance of customer parties.

      Click on this icon to delete a party.

      Click on this icon to modify the details of a party.

              Click on this button to add a new party.

      Click on this button to enter a Party from the Deal.

              Click on this button to return to the previous page without adding the participant.

Note: Prior to installing NetIMEX, a batch task was initiated by IMEX which identified all the parties found in existing deals related to the customer.  These parties comprise the basic list of parties included in your installed version.  When a transaction is opened containing a new party, the full name and address is entered.

Adding a New Party

To add a new party, click on the  button at the bottom of the Parties page.

The Party Details Maintenance page is displayed.

To complete the Party Details Maintenance section, fill in the relevant fields. Some of these fields are mandatory. They are indicated in Bold on the screen and by this icon   in the User Guide.

The Party Details Maintenance section requests the following information:

          Party ID - The code of the party as it appears in the participants list.

          Branch - The code of the branch.  NetIMEX allows the configuration of
several branches of the same party.

          Name - Full participant name.

          Street - Participant address.

          City - Participant location.

Country - Participant country

        Click on the save button to add the participant.

         Click on the cancel button to return to the previous page without adding the participant.

Changing Parties Details

To change the details of a party, click on the Edit  icon next to the Parties ID.

The Party details maintenance page is displayed.  The user may now update the Parties details (address fields).  The Party ID and Branch Code can not be changed.

Note: Only parties created by the customer may be edited.  The bank may modify parties downloaded from the host.

Deleting Parties

To delete a party, click on the  icon next to the Parties ID.

To view the Parties Profile window, select the Parties option from the Customize menu.

The Parties Profile window includes information about:

Party ID Code – The Participant’s code name. After receiving a transaction from the host, the Party ID is automatically added to the party profile.

Brach Name – The code of the participant branch.

Full Party Name – The participant’s full name.

Full Party Address – The participant’s full address.
