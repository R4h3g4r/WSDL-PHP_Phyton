General Bank User List

Click the General Bank Users link at the bottom of the Entities list in order to display the list of the General Bank Users in the system. The general bank User can perform only two actions in the system:

  • Create other General Bank Users.
  • Create Entities.

Note: There must be at least one General Bank User per Entity and the user will not be able to delete a General Bank User if he/she is the only one.

User ID - The User ID code by which the User is recognized in the System.

Delete  - Deletes the General User from the system, provided that he/she is not the only General User for that Entity.

Update - Enables the User to edit the General User Details.

 Click to add a General Bank User to the system.

 Click to return to the previous screen without saving any changes.

Click on the Entities link at the bottom of the General Bank Users List to return to the Entities List.
