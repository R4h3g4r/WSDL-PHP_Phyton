Import External File

This feature allows you to upload into NetIMEX a file containing prepared transaction created by a customer’s back-end system. 

The Import External File is available, according to the setup of the system, in the Import main menu.

When selecting this option from the Import Main Menu the Import External File Window is displayed. The file name, Transaction Type, Number of Transactions and Total Amount must be specified for validation purposes.

Text - Indicate the name of the .xml file to be imported.

Transaction Type - Select the type of transaction of the file to be imported. Currently two types of transactions are supported; Single Remittance and Multiple Remittances

Number of Transactions - Enter the total number of transactions in the file.

Total Amount - Enter the total amount of the transactions.

The Number of Transactions and Total Amount fields are used for validating the contents of the file and should match it.
