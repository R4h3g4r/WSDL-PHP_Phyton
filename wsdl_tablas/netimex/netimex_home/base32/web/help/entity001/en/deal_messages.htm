DEAL MESSAGES

Overview of Deal Messages

NetIMEX provides a Deal Message program exclusively for NetIMEX interactions.  The Deal Messages facility is provided for both import and export transactions.  Any message relating to an import transaction must be made from the Import Deal Messages Section.  Similarly, any message relating to an export transaction must be made from the Export Deal Messages Section.  Messages are stored permanently in IMEX’s Deal Messages section. Deal Messages can only be sent for existing deals.

Note: In NetIMEX, the Import and Export Deal Messages are used identically.  Therefore, in the following explanations, Import and Export are used interchangeably.

Upon selection of the Deal Message option on the Import menu the following Search Form is displayed.

The user can select a specific transaction by either:

  • Entering the Deal Number and/or Reference Number of the transaction that requires a Deal Message and clicking on the search button on the Search Form; or

  • By leaving the Deal Number and reference Number fields empty and clicking on the search button on the Search Form and selecting a specific transaction from the list of existing Deals that are displayed on the Deal Message Window.

Deal Messages can only be sent to the Bank relating to deals that have already been processed by the bank.  The Deal Message window displays a list of all deals, which do not have a message waiting to be sent.  Deals will not appear on the screen if they have open Deal Messages.

The list of all the Import items waiting for a response (Items in my in-box) includes:

  • Deals about which there is incoming correspondence from the Bank

  • Deals that are waiting for payment

  • Deals that are waiting for signatures

  • All transactions that are in process.

To view the deal, click on the deal title which is blue and underlined. No changes can be made to the deal from the view mode.

New Deal Message

To create a New Deal Message, click on the OPEN Icon  next to the deal you wish to amend. The selected Deal Message is displayed. The user enters all the relevant details of the message and clicks on the SAVE button.

This window contains a heading describing the deal which is the subject of the message.  Enter the Deal Messages title and message. Click on the Sign button to authorize the message.

The Message window also contains the fields:

Regular Deal Message or Urgent Deal Message Radio buttons – Allows the user to indicate the priority of the message.

Title - The tile given to the message.

Text – The free text message area.

Click on this button to send the message to the Transactions in Process section.

Erases the message.

Modify an Existing Message

To modify an existing Deal Message:

1.       From the Import menu, select Transactions in Progress by clicking on the Transactions icon or by clicking on the     Navigation Arrow  . The Transactions in Progress section appears.

 

2.       The Import Deal Messages Window appears. Select the Deal Message that you wish to modify and click on the Modify   icon; the Deal Message opens.

3.       Select the line describing the message you wish to modify.

4.       In the Title field, type in the title of the letter.

5.       If the letter is urgent, click Urgent Deal Messages.

6.       Type in the body of the text.

7.       Save and Sign the letter.
