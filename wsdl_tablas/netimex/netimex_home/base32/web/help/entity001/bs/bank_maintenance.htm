Bank Maintenance

Bank Maintenance is a function that is reserved for the Bank only to use in adapting and updating the functions to its specific requirements. Clicking on each topic link accesses each function.

The following areas can be edited in this section:

Group – Different Customers can be related by an umbrella organization known as a group. This section allows the user to view or edit the lists of customers and their super organizations.  

Group by Customer ID – This search field provides an alternative way of finding out to which group a customer belongs.

New Language – This function allows the Bank to define a new language in the system.

Delete Language – This function is used to delete a language in the system.

Tables Translation – The tables in the system originated in IMEX. This function translates IMEX tables to compatible formats in NetIMEX form.

Titles Translation – This function allows the Bank to edit any of the headings of program.

General Parameters – This function allows the Bank to customize different parameters that appear on each page.

Deal Number for Department – This function configures the deal number generation.

Help Page Configuration – This function specifies a Help page or Help application for each NetIMEX page different than the Help page delivered with NetIMEX making it unnecessary to override the supplied Help pages with the Bank’s Help pages.

Audit Trail – This function allows the Bank to configure activity logging and generate an inquiry regarding the activity in NetIMEX.

PO Integration – This function allows NetIMEX to perform operations related to acceptance of Purchase Orders from an external portal.

Set Up Dynamic Fields – This function allows the Bank to configure which of the Dynamic Fields for each transaction will appear as mandatory fields in NetIMEX.
