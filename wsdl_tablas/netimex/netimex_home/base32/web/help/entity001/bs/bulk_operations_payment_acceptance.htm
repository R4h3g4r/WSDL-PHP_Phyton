Payment / Acceptance - Bulk Operations

The user is able to process a number of selected Payment/Acceptance transactions in bulk.

How to handle a group of Payment/Acceptance transactions:

1.       Click on the Payment/Acceptance section header; the Bulk Payment/Acceptance Screen is displayed.

2.       On the Bulk Payment/Acceptance screen the user selects the relevant group of transactions that require either authorization or rejection, and clicks Search. If the user does not enter a value into any of the fields then, after clicking Search, a list of all the transactions waiting to be processed is displayed, see Figure 18 .

3.       In order to check all the transactions check the checkbox at the top of the column.

Next to each Payment, the user will find three icons that are used for Establishing Payment Transaction, Rejecting Payment Transaction and Printing Payment Transaction instructions.

4.       Click on Pay to display the new Bulk Payment /Acceptance page displaying details of all the selected transactions.

5.       Complete the required fields in the Bulk Payment/Acceptance Confirmation screen.

Note: As this screen operates in a bulk status, the Customer Instructions and Account Details will relate to all the payment/acceptance transactions selected.

6.       Click the appropriate button:
Save - To save all the transactions in the list.
Sign - To sign all the transactions according to the signature level.
Cancel - To return to the previous page.

Note: A user can only access the Bulk Payment/Acceptance function, according to the bank’s setup.
