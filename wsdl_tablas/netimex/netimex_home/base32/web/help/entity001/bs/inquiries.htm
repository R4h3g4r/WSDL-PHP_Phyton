INQUIRIES

Overview of Inquiries

This chapter includes details about making deal inquiries according to user-defined criteria. The following types of inquiries are explained in this chapter:

  • Transaction Status Inquiry - Inquiries about transaction status.
  • Open Deals Inquiry - Inquiries regarding transactions which are in process.
  • Deal History Inquiry - Inquiries relating to a specific transaction, including the original data and all transmissions relating to that transaction.
  • Deal Select Inquiry - General deal inquiries which are about selected groups of deals.
  • Inquiry Request - Report request inquires.

The Inquiries are listed in the Main Menu, which appears on the left side of the page. The Inquiry Request Form is displayed on the right side of the page as shown in the following screen.

To submit an Inquiry, click on the Inquiry link on Menu.
