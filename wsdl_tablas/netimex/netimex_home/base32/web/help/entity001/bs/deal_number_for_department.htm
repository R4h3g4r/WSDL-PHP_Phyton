Deal Number for Department

Updating the Department Numbers

From the department click on the Edit icon  .

The Deal Number for Department maintenance page is displayed.

The Deal Number for Department maintenance page displays a number of fields which show the current numerator for all transactions that can be initiated in NetIMEX: Import Letter of Credit, Guarantees, Remittance, Documentary Collections, Clean Collections.

Enter the new value for the required deal type(s) and click on the Save button 

Note: To specify that no deal number should be generated for a deal type, leave the deal type number field empty.

Viewing the Department Numerators

From the department list, click on the department name link.  The Deal Number for Department page is displayed.
