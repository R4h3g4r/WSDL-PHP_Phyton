Deal Select

The Deal Selection inquiry displays all deals that conform to the select parameters.

To view the deal selection window, choose the Deal Select option from the Inquiry menu or scroll down the Inquiry page. 

Deal selection inquiries are made according to the following criteria:

  • Deal Number
  • Reference or Invoice Number
  • Deal Type
  • Counterparty
  • Currency
  • Amount Range

Check the All Customers checkbox in order to activate the inquiry for a group of Customers. The availability of this field is dependent on the user having permission to activate inquiries at the group level.

To view a list of deals, specify the relevant parameters and click the Search icon.

To perform a general search to view all the deals sent to or received from the bank, click on the Search button without entering any of the fields.

Check the Save Criteria checkbox for NetIMEX to save the inquiry criteria as the default display in the user profile. All subsequent requests for the inquiry will use the saved criteria as the default.

If the user wants to remove a saved criteria status for the next default Transaction Status, thereby creating an empty criterion status, the following step needs to be taken:

  • All fields need to be blank, except the Save Criteria field, which need to be checked, and then click Search.

A user is allowed to use one criterion per inquiry.

To view a list of deals, specify the relevant parameters and click the Search icon.

To perform a general search to view all the deals sent to or received from the bank, click on the Search button without entering any of the fields.

To view a deal in greater detail, click on the individual Transaction in the Deal Type column.  The results of the search may be sorted by any of the heading topics found in the Table Heading. Double click on that column heading to perform a sort.
