NON RENEWAL GUARANTEE

Overview of a Non Renewal Guarantee

The renewal process is automatic. The outcome of the renewal is done in a back-end system and is then downloaded to NetIMEX. The user can instruct the Bank in case of non renewal.

Upon selection of the Non Renewal Guarantee option on the Standby/Guarantees menu the following Search Form is displayed.

The user can search for a specific transaction of a Non Renewal Guarantee in one of the following ways:

  • Entering the Deal Number and/or Reference Number of the Non Renewal Guarantee.
  • Entering a range of dates for the Ultimate Expiry Date.  
  • Leaving the Deal Number and Reference Number fields empty.

After selecting the criteria click on the Search button and then select a specific transaction from the list of Non Renewal Guarantee Applications.

The List of Non Renewal Guarantees includes deals where their Final Notice date has not yet arrived.

To view a deal, click on the deal title which is blue and underlined. No changes can be made to the deal from the view mode.

To retrieve a Non Renewal Guarantee in order to send the details back to the back-end system, click on the OPEN Icon  .
