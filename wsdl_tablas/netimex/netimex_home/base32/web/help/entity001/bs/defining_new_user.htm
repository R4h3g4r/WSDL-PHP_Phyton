Defining a New User

Defining a New User requires two separate steps. The first step in defining a New User is authorization and is performed by the Bank. The second step in defining a New User is assigning the User to the Customer and is performed by the Supervisor User at the Customer location.

A Group consists of a super organization that may contain several subdivisions that we define as Customers. Users belong to Groups and are authorized to enter transactions at the Bank on behalf of specific Customers.
