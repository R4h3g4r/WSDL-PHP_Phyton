Customer Detail

A user may handle Trade Finance transactions for a number of customers defined in the user's group. The field displays the customer on which the user is currently working.        

To change the customer the user is working on, click on the Customer link (i.e. the word customer at the top of the page is underlined).  A window with the customer the user is permitted to work on behalf of will be displayed.  Select the required customer and resume working.  

There is no Customer link (i.e. the word customer at the top of the page is not underlined) if the user is only allowed to work on one customer.
