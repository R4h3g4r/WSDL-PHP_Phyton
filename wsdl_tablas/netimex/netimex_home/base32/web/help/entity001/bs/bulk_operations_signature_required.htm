Signature Required - Bulk Operations

The user can sign a group of transactions (within the specific authorization levels) without the need to sign each item individually.

How to sign a group of signatures:

1.       Click on the Signatures Required section header; the Bulk Signature Criteria Screen is displayed.

2.       On the Bulk Signature Criteria screen the user selects the relevant group of transactions waiting for signature and clicks Search. If the user does not enter a value into any of the fields, a list of all the transactions waiting for signing is displayed.

3.       The Sum Total is a total of all the checked transactions on the list with the same currency. If more than one currency is checked no Sum Total is displayed.

4.        In order to check all the transactions check the checkbox at the top of the column.

5.       Click on the Sign button; a confirmation screen is displayed giving a summary of the transaction details.

6.       Enter your password and click on the Sign button.

Note: A user can only access the Bulk Signature Required function, according to the bank’s setup.
