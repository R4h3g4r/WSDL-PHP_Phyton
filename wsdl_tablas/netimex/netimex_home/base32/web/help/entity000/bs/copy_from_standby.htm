Copy From Standby

NetIMEX allows the user to initiate a new transaction based on an existing one even if it has not been released. By selecting the Copy From option the user can select either a transaction that was initiated in NetIMEX or the Bank’s authorized transaction relevant for the Product e.g. Import L/C.

How to copy an existing transaction:

1.       Select Copy From on the Inbox to display the search facility.

2.       In order to view all existing transaction leave all the fields empty and click on the search button.   
OR
3.       Search for a transaction by Deal Number, Transaction Name and/or Customer Reference and click on the search button; if it exists the relevant transaction is displayed.

4.       Click on the Copy From Icon to display the relevant NetIMEX application page which contains the data from the transaction it is based upon, excluding the deal and reference number. The data in the Establish By field indicates who created the transaction i.e. Customer (transaction initiated by the Customer) or Bank (transaction as authorized by the Bank).

5.        From now on the new transaction is completed like a regular transaction.
