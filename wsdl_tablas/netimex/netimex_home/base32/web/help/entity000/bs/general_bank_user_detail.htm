General Bank User

User ID

Clicking on the User ID field in the General Bank User list allows the user to view the User Details.

User ID - The User ID code by which the User is recognized in the system.

User Name - The full name of the User

Click  to return to the Bank General User List.

Add General Bank User

Click Add to create a new General Bank User. The User Details form is displayed. To complete the User Details form, enter the required information in the relevant fields. Both fields in this form are mandatory.

User ID - Enter the User ID code by which the User will be recognized in the system.

User Name - Enter the full name of the User

 Click to save the User in the system.

 Click to return to the Bank General User List without saving any changes.

Edit General Bank User Details

Click on the Edit icon to edit the User details. The User Details form is displayed.

The only field that can be edited on this form is the User Name. Enter a new name or change the existing name, as required.

 Click to save the changes to the User Name and return to the previous screen.

 Click to return to the General Bank User List without saving any changes.

Click on the Entities link at the bottom of the General Bank User list to return to the Entities List.

Regular Bank User Work Options with Entities

Customer

  • The regular Bank user can view and work with only those Customers defined in his Entity.
  • The regular Bank user can create new Customers and edit existing Customers only for his own Entity. The Entity for the newly created Customer will be set according to the Entity of the Bank User who created the Customer. The system will identify this Entity from the information entered at login.

Group

  • The regular Bank user can view and work with only those Groups defined in his Entity.
  • The regular Bank user can create new Groups and edit existing Groups only for his own entity. The Entity for the newly created Group will be set according to the Entity of the Bank user who created the Group. The system will identify this Entity from the information entered at login.

User

  • The regular Bank user can view and work with only those Users defined in his Entity.
  • The regular Bank user can create new Users and edit existing Users only for his own entity. The Entity for the newly created User will be set according to the Entity of the Bank user who created the User. The system will identify this Entity from the information entered at login.

Customization

The regular Bank user can customize certain features, for his Entity only. For details on customizing features, please refer to the section on Bank Maintenance.
