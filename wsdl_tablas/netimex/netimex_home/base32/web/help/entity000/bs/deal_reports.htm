Reports

Overview of Deal Reports

NetIMEX provides 2 ranges of reports which can be produced by the customer. To launch a Report, click on one of the headings in the Reports section of the Main Menu, which appears on the left side of the page.

  • Deal Report: Displays the individual amounts according to a selected criteria.
  • Deal Report Summarized: Displays the total amounts per customer according to a selected criteria.

Deal Report

When selecting the Deal Repot the Deal Report selection screen is displayed. This allows the user to select a report according to specific criteria.

The following criteria may be selected:

Report Name - Indicates that the report will display All Deals, Outstanding Deals or Closed Deals.

Deal Type - Indicates that the report will display all or specific deals supported by NetIMEX

Currency - Indicates that the report will display deals for any Currency allowed by the Bank.

    Date Range - Indicates that the report will display deals in the specified range of Issuance dates.

Local Currency - Indicates the currency in which the report results will be displayed.

If the Local Currency combo box is left empty, the local currency equivalent calculations will use the information contained in the Customer Local Currency field, if defined, from the Customize Customer screen.

    Order By - The user can select various High/Low or Low/High options, e.g. for currencies, Counterparties, Expiry Dates.

Do Sub Totals - When checked indicates that the report will display sub totals. Only relevant if Currency, Counter Party or Sight/Tenor Days is selected in the Order By field.

All Customers - When checked it indicates that all Customers will be displayed in the Deal Report. An additional column called “Customer Name” will be added to the report. This field is available depending on the setting in the Group Inquiries field in the Preference option.

 Click on the Preview button in order to display the report in an HTML format.

 Click on the Report button in order to display the finalized formatted report.
