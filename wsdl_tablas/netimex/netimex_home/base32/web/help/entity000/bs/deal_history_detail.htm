Deal History

To View Deal Details

The Deal History Inquiry enables you to view a deal in detail. To view a deal, select the Deal History option from the Inquiry menu. The Deal History Request Form appears.

Enter the deal number and/or your reference number (and deal type) as appropriate.

Save Criteria - Check this box for NetIMEX to save the inquiry criteria as the default display in the user profile. All subsequent requests for the inquiry will use the saved criteria as the default.

If the user wants to remove a saved criteria status for the next default Transaction Status, thereby creating an empty criterion status, the following step needs to be taken:

  • All fields need to be blank, except the Save Criteria field, which need to be checked, and then click Search.

A user is allowed to use one criterion per inquiry.

Click the Search button to view the deal.

 

The Deal Details appear together with a history of transactions performed on that deal. The deal history options enable you to view details entered by the customer and bank’s host.  This option is only active if you originated the transaction.  The transaction can be either an import or export transaction.

The following information may appear together with the deal details, depending on the nature of the deal:

  • Original Transaction Details (initiated by the customer)
  • Deal Participants
  • Settlement Data
  • Deal Texts
  • Deal Open Payment Details
  • Deal Open Acceptance Details
  • Deal Open Finances
  • Deal Open Margins

Note: The information that is available in the search results depends on the nature of the particular transaction.
