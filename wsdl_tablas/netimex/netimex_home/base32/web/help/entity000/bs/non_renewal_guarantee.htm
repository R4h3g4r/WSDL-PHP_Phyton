NON RENEWAL GUARANTEE

View a Non Renewal Guarantee

Navigation Tab

Clicking on any heading in the Navigation Toolbar from anywhere in the application form will bring you to that section.

The Direction Arrow or Page Numbers allow you to move through the deals.

There are two sections of the Non Renewal Guarantee that may be viewed.

  • General Details
  • Text

General Details

Clicking on the General Details heading in the Navigation Toolbar from anywhere in the application form will bring you to this section.

The following information is display:

Reference Number - A unique deal identification number, decided by the NetIMEX user, for example, a contract number, a delivery order.

Deal Number - The bank specifies the deal number once the guarantee Application has been received. When the application is returned, the deal number appears. 

Text

The second part of the Guarantee form contains the Text section. Scroll down from the General Details to access this section. You may also access the Text section by clicking on the Text heading on the yellow Navigation Toolbar.

This section enables you to review the following additional information relevant to the Guarantee:

Customer Instructions - Used to enter additional customer instructions.

Zoom Text Icon  - Click on this icon to enlarge the text box area. You may edit or enter new data in the enlarged text box.

Open Clause Library Icon  - When clicking on this icon a list of defined clauses relevant for the window are displayed. After selecting the required clauses the contents of the clause is copied to the window.

Clause Library
In order to display the clauses for a specific text window click on the Open Clause Library Icon next to the relevant window.

On the Clause Library window select the clause/s that should be entered into the text by checking the relevant checkbox and clicking on the Select button. In order to include all the Clauses listed in the Clause Library check the checkbox at the top of the column and click on the Select button.

When returning to the Text Screen the user can view the detailed description of the Clause which will be included in the Text. In order to remove a clause from the Text the user must delete the text of the clause from the relevant window.

Defined Clause Descriptions
An additional option exists to display clauses as defined in the clauses form, which the user can select by selecting the desired checkboxes. This replaces the use of the Clause Library and free text windows.

There are two options for this function, which is established during bank setup:

  • The user can select more than one clause from one text window.

  • The user can select only one clause from one text window.

In a text window which is marked as mandatory, a validity check is performed to ensure that at least one clause is selected.

Save and Authorize the Non Renewal Guarantee

The following commands are located at the bottom of the Non Renewal Guarantee window.

 Click here to save the data to the Non Renewal Guarantee application.

When the application is completed, it must be signed. Click here to sign the application.

This button returns the user to the previous page without saving any action on the current page.

When all the details have been entered into the application form, click on the Sign button. The system will run internal checks to make sure the information is entered correctly. If any errors exist, you will be notified about each error and given an opportunity to correct them.
