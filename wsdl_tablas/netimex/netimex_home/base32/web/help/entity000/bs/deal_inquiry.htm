Inquiry Request

This feature will enable customers to make a request for a back office generated report.  A back office generated report in this context means a back office document output which is not part of a deal step release, such as the Customer Limit Exposure report.  Click on the Inquiry Request option from the Inquiry section of the main menu. 

 

Note: When there are no entries available to be shown, the Inquiry Request option will not appear in the menu.

The Inquiry Request form will appear.

The available reports will be listed.  Click on the desired report and press Send.  The report request is sent to the Back Office and the result is viewed in the Incoming Correspondence section of the Inbox.
