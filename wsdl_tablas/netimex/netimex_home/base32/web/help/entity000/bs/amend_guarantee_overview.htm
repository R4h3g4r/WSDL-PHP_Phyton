AMEND GUARANTEE

Overview of Amending a Guarantee

Customer-initiated guarantees which have been transmitted to the Trade Finance department and subsequently issued by the bank are listed in the Amend Existing Guarantee Window. An application can be updated until it has been sent to the Bank. After that it must be amended.

Upon selection of the Amend Guarantee option on the Standby/Guarantees menu the following Search Form is displayed.

The user can select a specific transaction in order to amend a Guarantee Application by either:

  • Entering the Deal Number and/or Reference Number of the Guarantee that needs to be amended and clicking on the search button on the Search Form; or
  • By leaving the Deal Number and Reference Number fields empty and clicking on the search button on the Search Form and selecting a specific transaction from the list of existing Import Guarantee Applications that are displayed on the Amend Guarantee Window.

To view the deal, click on the deal title which is blue and underlined. No changes can be made to the deal from the view mode.

To retrieve a Guarantee in order to amend it, click on the OPEN Icon  next to the deal you wish to amend. The selected deal opens with its application form completed.
