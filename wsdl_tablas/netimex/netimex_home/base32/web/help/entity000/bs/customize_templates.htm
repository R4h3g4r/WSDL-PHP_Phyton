Templates 

This section allows the user to view, edit and delete deal templates.

All the templates that have been entered appear on the page in a list, segregated by transaction type. The menu all indicates which transactions contain templates by showing a template icon next to the activity.

To edit a template, click on the Edit   icon.

To delete a template, click on the Delete   icon.

To display a template, click on the link to that template indicated by a blue font and underlined.
