Transaction in Process - Bulk Operations

The user is able authorize at one time, a number of transactions that were previously completed but not yet authorized. These are the transactions visible in the Transactions in Process section of the Inbox.

1.       Click on the Transactions in Process section header, the Bulk Transactions in Process screen is displayed.

2.       On the Bulk Signature Criteria screen the user selects the relevant group of transactions and click Search. If the user does not enter a value into any of the fields, a list of all the transactions waiting for signature is displayed.

Note: The process of bulk operations on Transaction in Process is based on “correct” transactions. These are transactions that were saved without errors, which are displayed in the column Transaction Saved OK.

3.       In order to check all the transactions check the checkbox at the top of the column.

In addition the following icons are displayed next to each transaction:

  • Edit: Allows the user to edit the transaction.  If a transaction is being edited for a second time, the user will receive a warning that all previous signatures are about to be deleted.  Should this happen, the transaction will revert to “Transactions in Process” unless the user re-signs the edited transaction once again.

  • Print: Allows the user to print the current form.

4.       Click the appropriate button:

Sign - To sign all the transactions according to the signature level.
Delete - To delete all the transactions according to the signature level.
Cancel - To return to the previous page.

 

5.       Click on the Sign button, a confirmation screen is displayed giving a summary of the transaction details.

6.       Enter your password and click on the Sign button.

Note: A user can only access the Bulk Transaction in Process function, according to the bank’s setup.
