Help Page Configuration

NetIMEX can specify a Help page or Help application for each NetIMEX page different than the Help page delivered with Net IMEX making it unnecessary to override the supplied Help pages with the Bank’s Help pages.

The special Help Path item allows the Bank to specify where the Help page is located is so that the NetIMEX Help installation location can remain intact. In the Help Path specification a special token <Lang> defines if the new location is language sensitive (having a specific directory for each NetIMEX language defined) or it is a only one Language (in this case the <Lang> sub-path does not need to be specified in the Help Path specification).

An additional parameter in the General configuration page allows the Bank to specify whether Help pages are displayed in a separate window.
