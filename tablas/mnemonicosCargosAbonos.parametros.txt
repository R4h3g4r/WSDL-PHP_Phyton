Documentación de la tabla mnemonicosCargosAbonos.parametros
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
La tabla +mnemonicosCargosAbonos.parametros+ corresponde a una 
lista de mnemónicos que se utilizan en el módulo de Cargos y 
Abonos en desarrollos paralelos.

Cada fila de la tabla representa un mnemónico y la estructura de 
cada fila es la siguiente (es la misma estructura para todas las 
filas):

------------------------------------------------------------------
- La llave de la fila es el código del mnemónico (obligatorio).
- Debe seguir un campo +DESC+ con la descripción del mnemónico 
(obligatorio).
- Debe seguir un campo +OPERACION+ con el String "Cargo" o con 
el String "Abono", según corresponda al mnemónico (obligatorio).
- Debe seguir un campo +TIPOCUENTA+ con el String "Ahorro" o 
con el String "Corriente", según corresponda al mnemónico 
(obligatorio).
- Puede seguir un campo +PUERTAS+ con una lista separada por comas 
de alias de puertas. La idea es relacionar mnemónicos con perfiles 
de usuarios. Es decir, si el mnemónico tiene asociada una puerta y 
el usuario logueado también tiene la puerta, entonces el usuario 
sólo verá ese mnemónico. *EJEMPLO:* +PUERTAS=CARG_AHO,ABON_AHO;+ 
------------------------------------------------------------------
